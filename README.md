# Claradine House — Kite & Wing Base, West Wales

> For competent, independent kitesurfers & wing foilers. You ride. We handle everything else.

A booking website for a guided kite safari business based in Pendine, South West Wales.

## Stack

- Static HTML + CSS
- Inter & Playfair Display (Google Fonts)
- Fully responsive (mobile-first)
- No build step — open `index.html` or serve with any static server

## Quick start

```bash
# Serve locally
npx serve .
# Or just open index.html directly
```

## Structure

```
kite-project/
├── index.html    # Main site
├── styles.css    # All styles
└── README.md     # This file
```
